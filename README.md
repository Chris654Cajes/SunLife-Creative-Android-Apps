# SunLife Creative Android Apps

A collection of Android applications developed for SunLife financial advisors.

## Apps

### 1. SunLife Client Milestones
A gamification app for tracking client relationships and progress through points, streaks, and achievements.

- **APK**: `APK Files/SunLifeClientMilestones.apk`
- **Demo**: `Mobile Apps Demo/sunLifeClientMilestones.mp4`

### 2. SunLife Financial Planning
A comprehensive financial planning survey and management system for collecting client information.

- **APK**: `APK Files/SunLifeFinancialPlanning.apk`
- **Demo**: `Mobile Apps Demo/SunLifeFinancialPlanning.mp4`

### 3. SunLife Client Tactics
A rejection analysis and recovery strategy tool for insurance advisors.

- **APK**: `APK Files/SunLifeClientTactics.apk`
- **Demo**: `Mobile Apps Demo/SunLifeClientTactics.mp4`

## User Manuals

Each app includes a detailed user manual in the `Mobile App User Manuals` folder:
- `sunLifeClientMilestones.html`
- `SunLifeFinancialPlanning.html`
- `SunLifeClientTactics.html`

Each manual contains:
- App features overview
- Step-by-step user guide
- Video demo
- APK download link

## Installation

1. Download the APK file for the desired app
2. Transfer to your Android device
3. Enable "Install from unknown sources" in device settings
4. Open the APK file to install

## Requirements

- Android 7.0 (API level 24) or higher
- Minimum 50MB storage space
